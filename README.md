# crystal-shield-module

CrystalShield native module

## Installation

```sh
npm install crystal-shield-module
```

## Usage

```js
import CrystalShieldModule from "crystal-shield-module";

// ...

const result = await CrystalShieldModule.multiply(3, 7);
```
